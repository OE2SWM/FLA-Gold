# FLA Gold – Fragen Trainer (PWA)

Browserbasierte Lernapp (Progressive Web App).
Offline nutzbar nach dem ersten Laden, installierbar auf Android & iOS.

## Starten (lokal am PC/Mac)
1) Ordner entpacken
2) Im Ordner einen kleinen Webserver starten (wichtig wegen Service Worker)

### Python
- Windows:  python -m http.server 8000
- macOS/Linux:  python3 -m http.server 8000

Dann öffnen:
http://localhost:8000

## Installieren am Handy

### Android (Chrome)
1) App im Browser öffnen
2) Menü (⋮) → „App installieren“ / „Zum Startbildschirm hinzufügen“
3) Danach wie eine normale App starten (auch offline nach Erststart).

### iPhone/iPad (Safari)
1) App in Safari öffnen
2) Teilen-Symbol (□↑) → „Zum Home-Bildschirm“
3) Starten über das Icon.

## Hinweis
Wenn du die Dateien auf Webspace / GitHub Pages / Netlify hostest, ist die Installation noch einfacher.
